# AI コンテキスト作成

`$ARGUMENTS` で渡された対象ファイルまたはテーマについて、AI 作業向けの参照方針と指示ルールを作成してください。

## 引数

- 基本は `$ARGUMENTS` に対象ファイルパスを 1 つ以上渡す
- 指定されたファイルがある場合は、その内容を最優先で参照する
- ファイル指定がない場合のみ、`$ARGUMENTS` をテーマ文字列として扱う
- 明示のない `docs/99_tmp/` 全体は自動走査しない

## 手順

1. 対象ファイルがあれば確認し、何の作業で使う文脈かを明記する
2. 適用範囲と参照優先順位を整理する
3. 指示ルールと禁止事項を分けて書く
4. 更新条件と見直し契機を記載する
5. 関連ドキュメントがあれば参照を付ける

## 出力先

- `docs/90_ai_context/` 配下に保存する前提で Markdown 形式で出力する

## 出力フォーマット

- タイトル
- 目的
- 適用範囲
- 参照優先順位
- 指示ルール
- 禁止事項
- 更新条件
- 関連ドキュメント
