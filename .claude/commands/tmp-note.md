# 一時メモ作成

`$ARGUMENTS` で渡された対象ファイルまたはテーマについて、一時メモを作成してください。

## 引数

- 基本は `$ARGUMENTS` に元メモのファイルパスを 1 つ以上渡す
- 指定されたファイルがある場合は、その内容を最優先で参照する
- ファイル指定がない場合のみ、`$ARGUMENTS` をメモ化したい内容の文字列として扱う
- 明示のない `docs/99_tmp/` 全体は自動走査しない

## 手順

1. メモの目的と作業日を明記する
2. 現時点の情報を入力メモとして残す
3. 仮説と未確定事項を分けて書く
4. 次アクションと移管先候補を明記する

## 出力先

- `docs/99_tmp/` 配下に保存する前提で Markdown 形式で出力する

## 出力フォーマット

- タイトル
- 作業日
- 目的
- 入力メモ
- 仮説
- 要確認事項
- 次アクション
- 移管先候補
