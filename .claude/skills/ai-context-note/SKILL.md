# AI コンテキスト作成 skill

Claude Code の定型作業を `/ai-context-note` で呼ぶための skill。

## 使う場面

- AI 作業向けの参照優先順位を整理したい
- `docs/90_ai_context/` 向けの Markdown を作りたい
- 指示ルールと禁止事項を分けて残したい

## 入力の扱い

- 指定ファイルがある場合は、その内容を最優先で使う
- ファイル指定がない時だけ、会話中のテーマ文字列を元情報にする
- 明示のない `docs/99_tmp/` 全体は自動走査しない

## 進め方

1. 何の作業で使う文脈かを `目的` に書く
2. `適用範囲` と `参照優先順位` を整理する
3. `指示ルール` と `禁止事項` を分ける
4. `更新条件` と `関連ドキュメント` を残す

## 出力

- タイトル
- 目的
- 適用範囲
- 参照優先順位
- 指示ルール
- 禁止事項
- 更新条件
- 関連ドキュメント
