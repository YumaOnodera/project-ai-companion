# 比較検討 skill

`.claude/commands/compare-options.md` に対応する再利用用 skill。

## 使う場面

- 技術選定や運用方針を比較したい
- 比較表だけでなく推奨案も残したい
- `docs/10_research/` または `docs/40_decisions/` 向けに整えたい

## 入力の扱い

- 指定ファイルがある場合は、その内容を最優先で使う
- 比較対象は 3 つ以内に絞る
- 明示のない `docs/99_tmp/` 全体は自動走査しない

## 進め方

1. 比較テーマを 1 文で定義する
2. 比較軸を定義する
3. 各案を同じ粒度で評価する
4. 推奨案、採用理由、リスクと緩和策をまとめる

## 出力

- 比較テーマ
- 比較対象
- 比較軸
- 比較表
- 推奨案
- 採用理由
- リスクと緩和策
- 要確認事項
