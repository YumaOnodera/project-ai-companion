# ナレッジ化 skill

Claude Code の定型作業を `/knowledge-capture` で呼ぶための skill。

## 使う場面

- 会話や作業結果を再利用可能な知見へ整理したい
- 一時メモではなく `docs/50_knowledge/` 向けに整えたい
- 適用条件と非適用条件を分けて残したい

## 進め方

1. 元情報の要点を抽出する
2. 手順または判断基準に再構成する
3. `適用条件` と `非適用条件` を明記する
4. `注意事項` と `関連ドキュメント` を整理する

## 出力

- タイトル
- 更新日
- 概要
- 適用条件
- 非適用条件
- 手順 / 判断基準
- 注意事項
- 関連ドキュメント
