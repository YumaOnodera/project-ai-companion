# 文書表現レビュー skill

`.claude/commands/review-writing.md` に対応する再利用用 skill。

## 使う場面

- Markdown 文書の文体を整えたい
- 誇張表現や冗長表現を減らしたい
- 短い改善案をまとめて出したい

## 進め方

1. 冗長表現、誇張表現、AI っぽい定型句を確認する
2. `要確認` の付け忘れがないかを見る
3. 見出しや箇条書きの粒度を確認する
4. 問題箇所ごとに短い改善案を示す

## 出力

- 対象
- 指摘事項
- 改善案
- 要確認事項
