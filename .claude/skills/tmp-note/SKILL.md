# 一時メモ作成 skill

`.claude/commands/tmp-note.md` に対応する再利用用 skill。

## 使う場面

- 正式な保存先がまだ決まっていない
- 壁打ちや途中整理を `docs/99_tmp/` 向けに残したい
- 後で他の正本へ移管できるようにしておきたい

## 進め方

1. メモの目的と作業日を書く
2. 現時点の情報を `入力メモ` として残す
3. `仮説` と `要確認事項` を分ける
4. `次アクション` と `移管先候補` を明記する

## 出力

- タイトル
- 作業日
- 目的
- 入力メモ
- 仮説
- 要確認事項
- 次アクション
- 移管先候補
