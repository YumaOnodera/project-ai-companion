# 作業手順書作成 skill

`.claude/commands/workflow-note.md` に対応する再利用用 skill。

## 使う場面

- 定型作業を再実行しやすい手順書にしたい
- `docs/20_workflows/` 向けに整えたい
- 実行手順と確認ポイントを分けて残したい

## 進め方

1. 手順の目的と対象読者を明記する
2. 前提条件、必要権限、準備物を整理する
3. 実行手順を順番に書く
4. `確認ポイント` と `注意事項` を分ける

## 出力

- タイトル
- 目的
- 対象
- 前提
- 手順
- 確認ポイント
- 注意事項
- 関連ドキュメント
