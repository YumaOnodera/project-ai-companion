# Codex Configuration Templates

このディレクトリは、Codex 向け設定の雛形や補助資料を管理するための置き場です。

## 使い方

- `config.toml.example`: `~/.codex/config.toml` に反映するグローバル設定雛形
- `AGENTS.md.template`: プロジェクト指示の叩き台
- `skills/README.md`: リポジトリ内で管理する draft skill の扱い方

## 注意

- Codex はリポジトリ内 `.codex/` を自動読込しない
- 実運用のプロジェクト指示はリポジトリ直下の `AGENTS.md` を正本とする
- このディレクトリは「共有用の雛形と補助資料」の保管場所として扱う
