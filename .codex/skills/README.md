# Local Skill Notes

Codex のスキルは通常 `~/.codex/skills/` に配置します。

このディレクトリは、リポジトリ内で skill 案や対応表を管理したい場合の置き場です。

## 想定用途

- Claude の project custom command と対応する Codex skill の設計メモを残す
- 他リポジトリへ持ち込む前の draft skill を保管する
- 運用中の skill 一覧を簡潔に把握する

## このリポジトリで管理する draft

- `.claude/commands/` と対応する文書化系 skill を配置している
- 対応表は `skill-map.md` を参照する

必要に応じて各ディレクトリを `~/.codex/skills/` へコピーして利用します。
