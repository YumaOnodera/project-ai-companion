# Overview

## 1. この docs の対象

- 対象: 既存案件へ持ち込める AI 設定、文書運用ルール、補助資料
- 非対象: 案件固有の要件定義、仕様、設計、実装詳細

## 2. 現在の運用フォーカス

- Codex / Claude Code の持ち込み設定
- AI 作業向けの参照優先順位とガードレール
- 既存業務システム案件へ導入しやすいテンプレート整備

## 3. 参照順序

原則として以下の順で参照する。

1. `README.md`
2. `AGENTS.md`
3. `docs/README.md`
4. `docs/90_ai_context/*`
5. `docs/99_tmp/*`

## 4. 正本の定義

- AI 文脈と参照方針の正本: `docs/90_ai_context`
- 一時メモの置き場: `docs/99_tmp`

## 5. 更新原則

- 横断再利用できる最小限の内容だけを残す
- 事実と推測を分離して記述する
- 推測や未確定事項には `要確認` を付ける
- 同じ情報を複数箇所に持たない
